 🧠 NotebookLM Autonomous Processing Agent

An end-to-end **AI-powered autonomous automation agent** that processes multiple documents through **Google NotebookLM**, generates structured insights, and organizes outputs automatically.

---

## 🚀 Overview

This project is a **fully automated document processing pipeline** built using **Playwright + Python**, designed to:

- 📂 Read multiple files from a source folder  
- 🤖 Upload them to NotebookLM  
- 🧠 Trigger AI-powered Studio features  
- 📥 Download outputs into a destination folder  
- 🔁 Repeat for all documents automatically  

👉 This system combines **UI automation + intelligent interaction handling** to work with modern dynamic web applications.

---

## ⚙️ Key Features

### 📂 Multi-Document Automation
- Processes files from a **source directory**
- Executes tasks **sequentially and reliably**
- Supports scalable batch workflows

---

### 🤖 Autonomous NotebookLM Interaction
- Opens NotebookLM automatically  
- Creates new notebooks  
- Uploads documents (hybrid/manual-auto)  
- Navigates dynamic UI intelligently  

---

### 🧠 Studio Automation

Automatically triggers:

- 🎧 Audio Overview *(auto-generates)*  
- 🎥 Video Overview *(requires Generate click)*  
- 🧠 Mind Map  
- 📊 Infographic  
- 📋 Data Table  
- 📑 Flashcards  
- 📽️ Slide Deck  

❌ Skips unnecessary modules:
- Quiz  
- Reports  

---

### 📥 Output Management
- Downloads generated outputs  
- Stores in **destination folder**  
- Maintains naming consistency based on input file  

---

### 🧠 Hybrid Interaction Engine

To handle modern web UIs (React-based), the agent uses:

- ✅ Text-based DOM search  
- 🔥 JavaScript-triggered clicks  
- ⚠️ Coordinate fallback system  
- 🔁 Retry + delay handling
